(скачать архив, распаковать и в терминале запустить npm install или npm ci)

A)

- Отображает один вопрос за раз.
- Пользователь отвечает на вопрос, нажимает «Далее».
- Прогресс-бар показывает, сколько вопросов уже пройдено.
- Нельзя перейти к следующему вопросу, пока не выбран ответ.
- Все ответы сохраняются в общем состоянии (answers).

В App.vue хранится состояние, answers и surveyQuestions

Реализовать циклическое отображение вопросов через <component :is="...">.
Каждый вопрос должен эмитить единое событие:
emit('answer', { id: question.id, value })

Компоненты которые будут строится динамически хранятся в папке form.

Проанализировать приложение, понять что нужно сделать и решить задачу.

Т.е нужно самостоятельно добавить события где нужно и добавить динамический компонент component.

B)

Компонент SummaryPanel
выводит value из вопросов, а нужно чтоб выводил label, т.е нормальный текст.

(Когда выполните пункт A, поймете задачу)

C)

Форма создания вопросов (QuestionCreator) - создать компонент

Позволяет добавить новые вопросы в список surveyQuestions.

Поля:

id — уникальный идентификатор (строка);
title — заголовок вопроса;
description - описание вопроса;
type — тип вопроса (radio, checkbox, text, select);
placeholder - если type = text
options — массив вариантов ответов (type != text).

После добавления — новый вопрос появляется в общем списке и участвует в опросе.
